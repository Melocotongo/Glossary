Glosario

CSS (Cascading Style Sheets)
Lenguaje de estilos que se usa para dar diseño y formato a las páginas web (colores, fuentes, márgenes, etc.).
Discord
Plataforma de comunicación que permite chat de texto, voz y video, popular en comunidades de videojuegos y desarrollo.
Git
Sistema de control de versiones que permite gestionar y realizar un seguimiento de cambios en el código.
GitHub
Plataforma en línea que almacena repositorios de Git, facilitando la colaboración en proyectos de código.
HTML (HyperText Markup Language)
Lenguaje de marcado que estructura el contenido de una página web.
Etiquetas básicas de HTML
  • <html>: Define el inicio de un documento HTML.
  • <head>: Contiene metadatos y enlaces de estilos y scripts.
  • <body>: Contiene el contenido visible de la página web.
  • <h1>...</h1>: Títulos (hay de <h1> a <h6>).
  • <p>...</p>: Párrafo de texto.
  • <a href="url">...</a>: Enlace a otra página o sección.
  • <img src="ruta" alt="texto">: Imagen en la página.
IA (Inteligencia Artificial)
Campo de la informática que desarrolla sistemas capaces de realizar tareas que requieren inteligencia humana, como aprendizaje y reconocimiento de patrones.
IDE (Entorno de Desarrollo Integrado)
Software que proporciona herramientas para escribir, probar y depurar código (ej., Visual Studio Code).
Visual Studio Code
IDE ligero y popular, desarrollado por Microsoft, que admite múltiples lenguajes y extensiones para facilitar el desarrollo de software.
